# Password List Checker (Python)

This project reads a list of passwords from a file and evaluates their strength.

## Features
- reads passwords from a text file
- checks password length
- detects uppercase letters
- detects lowercase letters
- detects digits
- classifies passwords as Weak, Medium, or Strong

## Concepts used
- file handling
- loops (while, for)
- conditions (if)
- string processing

## Input file
Create a file named `passwords.txt` with one password per line.

## Execution
python pswrdcheck.py
